What is an API?

Api is an acronym for application programming interface. To understand that what is an api ,we first need to understand “what is an interface ? ”

Interface

An interface is a shared boundary across which two or more component of a system(computer system ) exchange information.Now the exchange can be between softwares, hardwares, humans, peripheral devices and combinations of these.

API(Application Programming Interface)

Api is an interface between the softwares where the information is exchanged(via programming) between two or more applications .So, it is the medium(or can be think of as a messenger) through which any two applications talk to each other.

One application will request for some information to the other application by sending some data .In response to that request the other application will provide the information to the application that requested it. And then the app will show that information in the readable format to the user. So, the medium(programming) through which the information is exchanged between these two applications is called an API(application programming interface) .

Real life examlple to understand "what is an api?"

Consider an example from the real life. Suppose that you go to the restaurant .There is you(user), table(the application through which you send the information), waiter( Api ) and kitchen(the other application from which the information will be fetched).

Now you call the waiter (i.e an api) and order him to bring masala dosa(provide the data to have some information i.e the menu in which masala dosa is selected) .The waiter carries the order(data) and give it to the kitchen(the other application retrieves the data).Then the kitchen provide the masala dosa ( provide the corresponding information)to the waiter(api) and waiter will carry that masala dosa back to the table(application that requests the information) and the user(end user) will consume that masala dosa(that fetched information).
